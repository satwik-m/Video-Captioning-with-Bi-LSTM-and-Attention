The drive link shared has all the required files including MSVD datasets, glove word embeddings, train-test split, saved models and generated results.

1. Create the directory structure like in the "MP" folder of the drive link shared.
2. Open Google colab in the browser and open the file with ".ipynb" extension.
3. Go to runtime tab and change the runtime to "GPU".
3. Set all the paths mentioned in the code according to your own directory structure.
4. The code for mounting the drive is also present.
5. Now run the cells one-by-one till train function call.
6. Once training is finished run the test code to generate results file in the drive.
7. You can find all the saved models and generated results in the drive.
